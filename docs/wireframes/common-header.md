# 共通ヘッダー ワイヤーフレーム

目的: すべての画面で共通して使うヘッダーのレイアウトを定義する。

レイアウト（上から横並び）:

- 左: ロゴ（クリックでダッシュへ）
- 中央: グローバルナビ（ダッシュ / 案件 / 請求）
- 右: 検索アイコン / 通知アイコン / ユーザーアバター（ドロップダウン：プロフィール・サインアウト）

ワイヤーフレーム（テキスト表現）:

--------------------------------------------------------------------------------
| ロゴ |           ダッシュ　案件　請求            | 🔍 🔔 [アバター] |
--------------------------------------------------------------------------------

レスポンシブ:
- モバイル: ハンバーガーメニューにナビを格納、検索はアイコンタップでフル幅検索バーを表示
