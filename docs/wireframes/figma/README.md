Figmaインポート手順

1. `docs/wireframes/figma/` にあるSVGファイルをダウンロードまたはリポジトリから取得します。
2. Figmaを開き、メニューから `File → Import` を選択するか、SVGをキャンバスへドラッグ＆ドロップします。
3. インポート後、各SVGは個別のフレームとして扱えます。必要に応じてレイヤーを分解して調整してください。

ファイル一覧:
- common-header.svg
- projects-list.svg
- dashboard.svg
- project-create.svg

注意:
- これらはワイヤーフレーム用のシンプルなSVGで、Figma上で微調整やコンポーネント化が必要です。
