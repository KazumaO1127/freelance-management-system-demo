# 案件登録（作成/編集）ワイヤーフレーム

目的: 案件を新規登録または編集するためのフォーム画面。

フォーム項目:
- タイトル（必須）
- クライアント名（必須）
- 単価（数値）
- 開始日 / 終了日（期間）
- ステータス（select: contact/negotiation/contracted/working/completed）
- メモ（textarea）

アクション:
- 保存（プライマリ）
- キャンセル（一覧へ戻る）
