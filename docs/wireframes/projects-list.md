# 案件一覧 ワイヤーフレーム

目的: 案件を一覧で確認し、検索・フィルタ・ページングで絞り込めるUI。

主要要素:
- ページヘッダー: タイトル（案件一覧） + 新規ボタン
- フィルタ行: キーワード検索、ステータス選択（contact/negotiation/contracted/working/completed）、クライアント名
- テーブル: カラム（ID / タイトル / クライアント / 単価 / 期間 / ステータス / 操作）
- ページネーション下部

ワイヤーフレーム（テキスト）:

案件一覧　[新規作成]

検索: [_____________] ステータス: [全て v] クライアント: [________]

--------------------------------------------------------------------------------

--------------------------------------------------------------------------------

各行の「編集」は案件編集画面へ遷移。
